To open the server:
    1. cd to backend
    2. type in terminal "node server.js"


To open the frontend:
    1.download vscod extention name "PHP Server"
    2. right click on CheckMeLanding.php
    3. select PHP Server: Serve project
    4. open browser with URL http://localhost:3000/PHPCodesForEachPages/CheckMeLanding.php

To open database:
    1. open MAMP server port 8889
    2. import sql file in database folder
